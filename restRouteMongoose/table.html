RESTful Routes

A table of all 7 RESTful routes

Name Path HTTP Purpose Mongoose Method
Index /blogs GET List all dogs Blog.find()
NEW /blogs/new GET Show new dog form N/A
Create /blogs POST create a new blog, then redirect somewhere Blog.create()
Show /blogs/:id GET Show info about one specific blog Blog.findById()
Edit /blogs/:id/edit GET Show edit form for one dog Blog.findById()
Update /blogs/:id PUT Update a particular blog, then redirect somewhere Blog.findByIdAndUpdate()
Destroy /Blogs/:id DELETE Delete a particular dog, then redirect somewhere Blog.findByIdAndRemove()
